# unity-ai-Eclipse

Plugin Unity qui permet de créer des objets 3D dans la scène via un chat IA.
Supporte **OpenAI** et **Gemini** avec switch.

## Fonctionnalités
- Chat IA réel (API)
- Commandes en français et en anglais
- Création d’objets 3D par texte
- Switch OpenAI / Gemini
- Installable via Git URL

## Installation
Unity → Window → Package Manager  
→ Add package from git URL  
→ https://github.com/TON_USER/unity-ai-chat.git

## Configuration
Définir les clés API en variables d’environnement.
